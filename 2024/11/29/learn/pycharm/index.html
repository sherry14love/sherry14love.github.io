Pycharm食用指南

2024-11-29 2025-02-24
Toollearn

快捷键指令：

代码重构与质量检测

  • Ctrl + Alt + Shift + T: 打开重构菜单
  • Ctrl + Alt + L: 自动格式化代码
  • Alt + Enter: 查看代码检查修复建议

文件搜索与快速导航

  • Ctrl + N: 搜索类名
  • Ctrl + Shift + N: 搜索文件
  • Ctrl + E: 查看最近打开的文件

自动补全与重构快捷键

  • Alt + Enter: 修复错误
  • Shift + F6: 重命名变量或函数

代码块与行操作

  • Ctrl + Y: 删除当前行
  • Ctrl + D: 复制当前行
  • Ctrl + /: 快速注释/取消注释

代码管理与调试功能

  • Shift + F9: 启动调试模式
  • F8: 逐步执行代码
  • F9: 跳至下一个断点

编辑器操作

  • Ctrl + Z: 撤销
  • Ctrl + Shift + Z: 重做
  • Ctrl + X: 剪切当前行或选中的内容
  • Ctrl + C: 复制当前行或选中的内容
  • Ctrl + V: 粘贴
  • Ctrl + Shift + V: 从剪贴板历史中选择粘贴内容
  • Ctrl + W: 扩展选中范围
  • Ctrl + Shift + W: 缩小选中范围
  • Ctrl + Shift + U: 大小写转换
  • Ctrl + Shift + J: 合并行
  • Ctrl + Shift + Enter: 补全当前语句

导航与搜索

  • Ctrl + B: 跳转到声明（变量、函数、类等）
  • Ctrl + Alt + B: 跳转到实现
  • Ctrl + Shift + B: 跳转到类型声明
  • Ctrl + U: 跳转到父类或父方法
  • Ctrl + Alt + Left/Right: 导航回退/前进
  • Ctrl + Shift + Backspace: 跳转到上次编辑位置
  • Ctrl + Shift + F7: 高亮显示当前文件中的变量
  • Ctrl + Alt + F7: 显示使用情况

代码生成与重构

  • Alt + Insert: 生成代码（构造函数、getter/setter、toString等）
  • Ctrl + Alt + M: 提取方法
  • Ctrl + Alt + V: 提取变量
  • Ctrl + Alt + F: 提取字段
  • Ctrl + Alt + C: 提取常量
  • Ctrl + Alt + P: 提取参数

调试

  • Ctrl + F8: 切换断点
  • Ctrl + Shift + F8: 查看所有断点
  • Alt + F8: 在调试时计算表达式
  • F7: 进入方法内部
  • Shift + F7: 智能进入方法内部
  • F9: 恢复程序执行
  • Ctrl + F2: 停止调试

版本控制

  • Ctrl + K: 提交更改
  • Ctrl + T: 更新项目
  • Alt + `: 打开版本控制操作菜单

其他

  • Ctrl + Alt + S: 打开设置
  • Ctrl + Alt + Shift + S: 打开项目结构
  • Ctrl + Shift + A: 查找操作
  • Ctrl + Tab: 切换编辑器选项卡
  • Ctrl + Shift + F12: 最大化编辑器窗口
